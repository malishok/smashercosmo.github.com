Автосалонам

Возможности бесплатного сервиса unicom24.ru для автосалонов:

Кредитование

  • - Кредитный калькулятор с подробным описанием продукта и графиком платежей по всем банкам
  • - Заполнение одной анкеты, и нажатием «одной кнопки» заявка на автокредит отправится во все банки-партнеры (без использования эл. почты)
  • - Полностью прозрачная история прохождения заявки по статусам между автосалоном и банком в режиме реального времени
  • - Сохранение анкеты клиента на любом этапе заполнения
  • - Универсальная форма согласия клиента на обработку персональных данных
  • - "Обратная связь" с банком: в режиме реального времени редактирование отправленных заявок, согласно комментариям банка, без телефонных звонков
  • - Управление продажами и командой в режиме реального времени. Руководитель компании-партнера может отслеживать статусы заявок в банк в разрезе сотрудников, а также оформление страховок и пролонгацию

Страхование

  • - Единый страховой калькулятор по всем страховым компаниям
  • - система «Электронных полисов»: автоматическая фиксация всех статусов прохождения бланков строгой отчетности, отчеты по остаткам БСО, детальная статистика показателей по продажам
  • - Внесение данных по клиенту один раз, и оформление любого вида страхования (КАСКО, ОСАГО, ДГО и др.) любой страховой компании, нажатием «одной кнопки»
  • - Оформление пролонгации полиса: данные по клиенту перенесутся автоматически в новый полис из прошлого периода
